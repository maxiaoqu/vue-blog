## 安装
```
npm install
```

## 运行
```
npm run serve
```

## 打包
```
npm run build
```

## 检查
```
npm run lint
```

## 文件结构
```shell
├── public...........................打包所需静态资源
└── src..............................存放文件的相关目录
    ├── assets.......................项目静态资源
    ├── components...................模板组件
    ├── dictionary...................字典匹配库
    ├── libs.........................公用的方法库
    ├── router.......................路由文件
    ├── store........................vuex状态管理
    ├── view.........................vue页面文件
    ├── APP.vue......................app.vue文件
    └── main.js......................vue相关配置
```
